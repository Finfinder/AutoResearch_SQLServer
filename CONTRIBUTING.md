# Contributing to AutoResearch SQL Server

Thank you for your interest in improving AutoResearch SQL Server.

## Issues and Roadmap

- Report bugs and feature requests through [GitHub Issues](https://github.com/Finfinder/AutoResearch_SQLServer/issues).
- Larger goals are tracked with milestones and the pinned `Project Status / Roadmap` issue.


## Commits and Pull Requests

- Link related work in commits and pull requests with `Refs #123`, `Fixes #123`, or `Closes #123`.
- For pull requests targeting version branches, prefer `Refs #123` unless the branch change is intended to fully close the issue there.
- Keep commit subjects aligned with this repository's local conventions: short English imperative subjects without `feat:` or `fix:` prefixes.
